Install Nethunter on Oneplus 8T

June 17, 2024
Hacking, Android
Kali, Linux, Android, Hacking

Step 1. Download prerequsitions. #

CrDroid-8.7-kebab-ROM

payload-dumper-go

Nethunter-crdroid-overlay

Nethunter-crdroid-kernel

Magisk

TWRP-recovery

Step 2. Flash crdroid rom #

Reboot into fastboot and exec the following command.

unzip crDroidAndroid-12.1-20220728-kebab-v8.7.zip
tar xzvf payload-dumper-go*.tar.gz
chmod +x ./payload-dumper-go*
./payload-dumper-go* payload.bin
cd extracted_*_*
ls -1 *.img | sed s/.img//g | xargs -I {} fastboot flash {} {}.img

If you encounter some error, you might need to disable oem lock by

fastboot oem unlock

And if there are errors about not enough disk space left, you should delete some partitions.

fastboot delete-logical-partition .*-cow

Remove the partitions end with cow or start with my. Then you can proceed with fastboot flash again.

All done, reboot into system and setup the OS.

Step 3. Install Magisk #

  • reboot into fastboot and flash TWRP
  • fastboot reboot into recovery and select update from adb
  • adb sideload Magisk.apk
  • reboot into system to setup Magisk (it might require you to reboot multiple times)

Step 4. Install kali nethunter #

  • adb push overlay and kernel to /sdcard/
  • open Magisk to install Nethunter-crdroid-overlay wait until it finish
  • reboot into system again and update apps through nethunter f-droid
  • open nethunter app -> install kernel -> select Nethunter-crdroid-kernel
  • after install, reboot into system again.

Step 5. Have fun #

Now you can put the internal wifi interface into monitor mode. But nothing more than that.
