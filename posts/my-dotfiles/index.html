Archlinux Customization Full Guide

June 20, 2024
Linux
Archlinux, Windows Manager, Distros, Minimalism

archlinux minimal installation #

  • grab image file
  • nmctl connect to internet
  • archinstall
    • luks, no need for greeter

setup users and package manager #

  • pacman -S base-devel git
  • git clone https://aur.archlinux.org/yay.git
  • makepkg -si

install hyprland or sway #

  • yay -S hyprland sway swayidle swaylock wmenu librewolf-bin nerd-fonts zsh wl-clipboard grim artem wireguard-tools mpv neovim fastfetch btop slurp dmenu-wl brightnessctl hyprlock
  • nvchad oh-my-zsh wl-recorder gdb gcc btop ranger
  • libvirt
  • optional: wofi, ripgrep jq mat2 rofi imv feh httpie wf-recorder wofi dmenu-wl wmenu

replace swaybar with waybar #

  • yay -S waybar
  • comment out your current swaybar in your sway config and add this line “bar swaybar_command waybar”

#

install login manager tui-greet and screenlock swaylock #

  • systemctl enable greetd
  • configure /etc/greetd/config.toml example conf
command = "tuigreet --cmd sway -t -g '               +..,.          \n          ++      +,s,.       \n        .sssssss.    ,ss,     \n     +,ssssss,.       .sss+   \n   .sssssssss,+        ,sss+  \n   +,sss,.+,sss,.      .sss,  \n      .+     .ssss,+   ,sss,  \n               .,ssss,,ssss.  \n      ,ssss,.++  +,sssssss.   \n  +.,sss,.,sssssssssssssss,+  \n.sssss+     +.,,,,,,,.+,sssss.\nssss.                    .ssss\nCyka Blyat!!!\n' --theme 'text=red'"
