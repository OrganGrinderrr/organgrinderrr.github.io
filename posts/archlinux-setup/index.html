archlinux minimal installation #

  • grab image file from arch boot website
  • proceed to mirror config
    • ctrl + c and cat /etc/pacman.d/mirrorlist
    • select your nearest node, eg http://uk.mirror.archlinuxarm.org/$arch/$repo
  • partitioning, go with default scheme
  • install package fully automatic
  • install boot loader

setup users and package manager #

  • add user to sudoer
  • change hostname
  • ln -s /usr/bin/nvim /usr/bin/vi
  • install yay from source

install hyprland or sway #

  • yay -S hyprland-git
  • UTM have to switch to gl hardware
  • configure hyprland, resolution,scale and refresh rate

install login manager tui-greet #

  • systemctl enable greetd
  • configure /etc/greetd/config.toml example conf
